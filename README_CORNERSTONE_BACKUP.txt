# Backup Manual - Cornerstone Delivery App

Este arquivo descreve como atualizar o backup do aplicativo exportado do Bolt e armazenado no GitHub.

## 📁 Caminho da pasta local
~/Downloads/cls app backups/SCHEDULE PLANNER - APP/project

## ✅ Primeira vez (já feito)

1. Entrar na pasta do projeto:
    cd ~/Downloads/cls\ app\ backups/SCHEDULE\ PLANNER\ -\ APP/project

2. Iniciar o repositório Git:
    git init

3. Adicionar arquivos:
    git add .

4. Fazer commit:
    git commit -m "Primeiro backup do app exportado do Bolt"

5. Conectar ao GitHub:
    git remote add origin https://github.com/DupinsDigitals/cornerstone-delivery-app.git

6. Enviar para o GitHub:
    git push --force origin main

---

## 🔁 Atualizando o backup (futuro)

1. Entrar na pasta do projeto:
    cd ~/Downloads/cls\ app\ backups/SCHEDULE\ PLANNER\ -\ APP/project

2. Adicionar mudanças:
    git add .

3. Fazer novo commit:
    git commit -m "Atualização do backup com nova versão do app"

4. Enviar para o GitHub:
    git push --force origin main

---
